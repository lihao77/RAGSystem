# GeoPLUS 专题文档

本目录用于存放 GeoPLUS 扩展专题文档。它属于仓库的扩展专题，不属于当前主线架构入口，也不是历史归档区。

GeoPLUS 采用与主线仓库锁步的 `Plus` 仓库模式：

- `RAGSystem`：主线平台仓库
- `RAGSystem-GeoPLUS`：Geo 增强仓库

## 当前专题文档

- [REPO_GOVERNANCE.md](REPO_GOVERNANCE.md) — GeoPLUS 仓库治理与协作约定
- [MAINLINE_EXTENSION_POINTS.md](MAINLINE_EXTENSION_POINTS.md) — 主线仓库可扩展点说明

## 使用边界

- 本目录用于 GeoPLUS 扩展专题，不承担仓库主线架构导航职责。
- 主线系统文档请返回 `docs/README.md` 或对应子项目 `docs/README.md`。
- 若 GeoPLUS 历史方案不再作为当前专题维护，应迁移到 `docs/archive/`，而不是继续留在本目录。
